Package Source

Hedgehog is designed to deploy packages from a package source. A package source is typically a feed in ProGet, but it can be any webservice that implements the Universal feed API.

To add or manage package sources, go to Admin > Package Sources. Each package source consists of:

  • Feed URL - the API endpoint URL
  • Name - the name displayed
  • Credentials - optional API Key or username/password to authenticate to the feed

Package sources aren't an absolute necessity of Hedgehog; you can use an attachment instead.
