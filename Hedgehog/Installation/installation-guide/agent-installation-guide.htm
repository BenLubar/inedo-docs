To facilitate communication between Otter and the Windows servers you want to configure and orchestrate, Hedgehog uses a lightweight agent with a highly-optimized and resilient protocol. Installing agents is even easier than installing Hedgehog, and the Agent Installation Guide will provide step-by-step instructions, as well as provide some detail as to what's happening behind the scenes.

Shared Documentation Note - see the Agent Installation documentation for the agent installation details, and see the Hedgehog specific, Adding the Server to Hedgehog, below:

Adding the Server to Hedgehog

Once the agent has been installed, the server must be added to Hedgehog. Click the Create Server button on the Servers page, and enter the host name of the server, the port you installed the agent on, and the encryption key if you chose one.
